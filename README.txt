# A simple python script to create password

# To reach below help you can run: python main.py --help from cmd

usage: main.py [-h] [-q LENGTH] [-u] [-l] [-d] [-p]

Creating password is easy

optional arguments:
  -h, --help            show this help message and exit
  -q LENGTH, --length LENGTH
                        length of password
  -u, --upper           uppercase letters
  -l, --lower           lowercase letters
  -d, --digits          digits are in!
  -p, --pun             punctuations are in
